### 开发工具
- **测试框架**: Playwright

## 🚀 部署说明

### 环境要求
- Node.js >= 20.x
- npm >= 8.x
- HBuilderX（可选，用于 uniCloud 配置）

### 安装步骤
```bash
# 克隆项目
git clone git@github.com:iyuenan3/OpenClaw-PetsLog.git
cd OpenClaw-PetsLog

# 安装依赖
npm install

# 配置 uniCloud
# 在 HBuilderX 中关联云服务空间

# 上传云函数
# 右键 uniCloud-aliyun 目录 → 上传所有云函数

# 启动开发服务器
npm run dev:h5        # H5 端
npm run dev:mp-weixin # 微信小程序端

# 构建生产版本
npm run build:h5
npm run build:mp-weixin
```
