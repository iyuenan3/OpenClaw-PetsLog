## ❌ 失败的测试（39 个）

### 主要失败原因分析

#### 1. 页面标题/选择器不匹配
**影响文件**: login.spec.js, medication.spec.js, weight-record.spec.js, reminder.spec.js, search.spec.js

**典型错误**:
```
Error: expect(page).toHaveTitle(expected) failed
Expected pattern: /PetsLog/
Received string: "登录"
```

**原因**: 
- 页面标题与实际不符
- CSS 选择器与实际 DOM 结构不匹配
- 页面元素 class 名变更

#### 2. 新增功能页面未正确路由
**影响文件**: medication.spec.js, reminder.spec.js, search.spec.js

**典型错误**:
```
Error: locator('text=用药记录').toBeVisible() failed
element(s) not found
```

**原因**:
- 新增页面可能未正确配置路由
- 页面标题设置不正确
- 元素选择器需要更新

#### 3. 认证状态未正确复用
**影响文件**: 所有使用 chromium-auth 的测试

**原因**:
- 认证文件路径可能不正确
- 登录状态未保持
