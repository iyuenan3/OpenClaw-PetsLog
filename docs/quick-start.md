## 🔧 常见问题

### Q: 启动失败？
```bash
# 清理依赖重新安装
rm -rf node_modules package-lock.json
npm install
```

### Q: 语音输入没反应？
- 检查浏览器权限（Chrome/Edge 支持最好）
- 确认麦克风已授权
- 尝试使用 Chrome 浏览器

### Q: 离线模式不工作？
- 检查是否真的断网（飞行模式）
- 查看浏览器 Console 是否有错误
- 确认 localStorage 可用

### Q: 云函数调用失败？
- 确认已关联 uniCloud 服务空间
- 检查云函数是否上传
- 查看网络请求错误信息
