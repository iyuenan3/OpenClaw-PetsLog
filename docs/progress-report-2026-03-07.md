## 🎯 新增功能详情（2026-03-07）

### 离线模式 📴

**功能特性**:
- 实时网络状态检测（WiFi/4G/5G）
- 离线时数据自动保存到本地缓存
- 网络恢复后一键同步所有离线数据
- 页面顶部显示网络状态提示

**技术实现**:
```
src/utils/network.js        - 网络状态检测工具
src/utils/offline-cache.js  - 离线缓存管理工具
src/App.vue                 - 全局网络监听
```

**使用场景**:
- 地下室、电梯等无信号区域
- 飞行模式使用
- 网络不稳定环境

---

### 语音输入 🎤

**功能特性**:
- H5 端：Web Speech API 实时语音识别
- 微信小程序：录音功能支持
- 智能解析：支持"3.5 公斤"、"三点五 kg"等表达
- 按住说话，松开发送

**技术实现**:
```
src/components/VoiceInput.vue   - 语音输入组件
src/pages/weight/add-weight.vue - 体重记录页面（集成语音输入）
```

**使用示例**:
- "3.5 公斤" → 自动识别为 `3.5`
- "三点五 kg" → 自动识别为 `3.5`
- "体重三点五千克" → 自动识别为 `3.5`

## 🔧 技术亮点

### 1. 跨平台网络检测
```javascript
// 微信小程序
wx.onNetworkStatusChange((res) => { ... })

// H5 端
window.addEventListener('online/offline', handler)
```

### 2. 离线缓存策略
- 队列式缓存：按时间顺序保存操作
- 批量同步：网络恢复后一次性同步
- 失败重试：同步失败的数据保留在缓存中
- 容量限制：最多 100 条，防止存储溢出

### 3. 语音识别解析
```javascript
// 支持多种表达
"3.5 公斤" → 3.5
"三点五 kg" → 3.5
"体重三点五千克" → 3.5

// 正则提取 + 中文数字映射
const match = text.match(/(\d+\.?\d*)/)
```
