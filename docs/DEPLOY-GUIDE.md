## 📋 快速开始

### 1. 环境准备

#### 前置要求
```bash
# Node.js >= 16.x
node -v  # 应该显示 v16+

# npm >= 8.x
npm -v   # 应该显示 8+
```

#### 安装依赖
```bash
cd petslog-clean
npm install
```

---

### 2. 启动开发服务器

```bash
# H5 开发模式
npm run dev:h5

# 访问地址
http://localhost:5173
```

---

### 3. 首次使用

#### 注册账号
1. 打开 http://localhost:5173
2. 点击"注册"按钮
3. 输入用户名和密码
4. 完成注册

#### 添加宠物
1. 登录后进入首页
2. 点击右上角"+ 添加"
3. 填写宠物信息：
   - 名字
   - 物种（猫/狗）
   - 品种
   - 性别
   - 出生日期
   - 毛色
   - 绝育状态
4. 上传宠物头像（可选）
5. 保存

---

## 🎯 核心功能使用

### 1. 用药记录 ⭐（乔治专用）

#### 添加用药记录
1. 进入"用药记录"页面
2. 选择宠物（如：乔治）
3. 点击"+"添加
4. 填写信息：
   - 药品名称：苯巴比妥片
   - 剂量：1 片
   - 频率：每天 2 次
   - 用药目的：控制癫痫
   - 用药说明：饭后服用
   - 启用提醒：打开
   - 提醒时间：08:00, 20:00
5. 保存

#### 用药打卡
1. 找到对应记录
2. 点击"✅ 已用药"
3. 系统自动计算下次用药时间

---

### 2. 体重记录（离线 + 语音）

#### 手动添加
1. 进入宠物详情页
2. 切换到"体重"Tab
3. 点击"添加体重记录"
4. 输入体重（如：3.5kg）
5. 选择日期
6. 添加备注
7. 保存

#### 语音输入（H5 端）
1. 点击"🎤 按住说体重"
2. 按住按钮
3. 说"3.5 公斤"
4. 松开按钮
5. 验证识别结果
6. 保存

#### 离线模式
1. 开启飞行模式
2. 添加体重记录
3. 数据自动保存到本地
4. 关闭飞行模式
5. 点击"🔄 同步离线数据"

---

### 3. 提醒功能

#### 查看提醒
1. 进入"提醒中心"
2. 查看所有待处理提醒
3. Tab 切换分类：
   - 全部
   - 驱虫
   - 疫苗
   - 生日
   - 称重

#### 标记完成
1. 找到一条提醒
2. 点击"✓"标记完成
3. 系统自动计算下次提醒时间

#### 提醒设置
1. 进入"提醒设置"
2. 启用/禁用各类提醒
3. 设置提前提醒天数
4. 设置通知时间
5. 设置免打扰时段
6. 保存

---

### 4. 搜索功能

#### 搜索宠物
1. 首页顶部搜索框
2. 输入宠物名字或品种
3. 实时显示搜索结果

#### 筛选宠物
1. 点击筛选标签：
   - 全部
   - 🐱 猫咪
   - 🐶 狗狗
2. 快速筛选

## 🐛 常见问题

### Q: 语音输入没反应？
**A**: 
- 检查浏览器权限（Chrome/Edge 支持最好）
- 确认麦克风已授权
- 尝试使用 Chrome 浏览器

### Q: 离线模式不工作？
**A**:
- 检查是否真的断网（飞行模式）
- 查看浏览器 Console 是否有错误
- 确认 localStorage 可用

### Q: 提醒不显示？
**A**:
- 确认云函数已上传
- 检查定时任务配置
- 查看通知设置是否启用

### Q: 用药提醒时间不对？
**A**:
- 检查频率设置是否正确
- 验证时区设置（Asia/Shanghai）
- 重新保存用药记录
