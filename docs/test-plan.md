## 🚀 测试步骤

### 准备工作
1. 克隆项目：`git clone git@github.com:iYuenan3/OpenClaw-PetsLog.git`
2. 安装依赖：`npm install`
3. 配置 uniCloud：关联云服务空间
4. 启动开发服务器：`npm run dev:h5`
5. 打开浏览器访问：`http://localhost:5173`

### 测试顺序建议
1. **基础功能**: 登录 → 宠物列表 → 宠物详情
2. **核心功能**: 体重记录（含语音输入、离线模式）
3. **记录功能**: 健康/驱虫/疫苗/粮食
4. **增强功能**: 提醒中心 → 数据统计 → 数据导出
5. **UI 功能**: 深色模式 → 主题切换
